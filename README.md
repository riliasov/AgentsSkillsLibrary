# Agents Skills Library

Этот репозиторий содержит базу знаний и инструкции для AI-агентов.

## Структура
- `Agents-skills/` — основная папка со скиллами, ролями и командами.
- `README.md` — этот файл.
- `CHANGELOG.md` — история изменений структуры и скиллов.

## Использование в проектах
Для подключения библиотеки к проекту используйте симлинк на внутреннюю папку:

```bash
ln -s ~/Developer/AgentsSkillsLibrary/Agents-skills .AgentsSkillsLibrary
```

Это позволит агентам находить инструкции по пути `.AgentsSkillsLibrary/00_AGENTS_FIRST.md`.
